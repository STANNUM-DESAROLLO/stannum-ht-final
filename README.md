# STANNUM — IA Summit Tucumán 2026

Landing del evento **IA SUMMIT TUCUMÁN 2026** (17 de junio · Hilton Tucumán).

- Sitio en vivo: https://stannum-ht-final.vercel.app/
- Stack: HTML estático + CDN (GSAP/ScrollTrigger, Lenis, Spline 3D, Three.js).
- Robot 3D del hero: embebido desde Spline (`prod.spline.design`), no se edita en este repo.

## Estructura
- `index.html` — landing completa (estilos y JS inline).
- `agencia-g.svg` — logo Agencia G (co-producción).

## Deploy
Conectado al proyecto Vercel `stannum-ht-final`. Un push a `main` publica en producción.
Trabajar en rama `cambios-junio` y mergear a `main` cuando esté aprobado.
